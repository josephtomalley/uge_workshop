4 traits thats make unix developer friendly:
  -unix data is store as plaintext files
  -nested subdirectories
  -made of many small programs
  -input-output redirectionn(input from some place besides the keyboard and output to somewhere besides the screen)

Unix kernel:
  -program that manages hardware
  -CPU/memory/devices->kernel->other programs
  -the kernel is safe from hackers
  -
Unix shell:
  -program that provides a user interface.
  -GUI/terminal/touch/audible
  -prevents the user from directly accessing the kernel
